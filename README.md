 * [SonarQube](https://sonarcloud.io/dashboard?id=com.asteroid.duck%3Ajena-utils) - Code quality and test coverage
 * [Bintray](https://dl.bintray.com/duck-asteroid/maven/) - Hosts binary releases (Maven2 repo)
